# Hackspot 🚀

# How to use this project 🛠️

Run the following commands from your terminal👇🏻

1) Clone repository 🖇️ 
```
git clone https://github.com/Lamia-Abdullah/Hackspot.git
```
2) Install all the required dependencies ⏳
``` 
flutter pub get
```
3) Run the app ✨
```
flutter run
```
